Save my Tabs - Settings

 

Auto-save

Options in this page do not affect main functionality accessed through the menu popup.

 

   Enable/Disable

Enable/disable automatic tab saving functionality.

When enabled, the addon will periodically take a snapshot of all open tabs across all open browser windows and save them in a daily bookmark subfolder under AUTOSAVE.

Daily bookmark subfolder has title in YYYY-mm-dd format.

 

   Interval

This option controls the interval (in minutes) at between auto-save snapshots.

The timer gets reset every time settings are updated, saved, or the first browser window gets opened.

 

   Overwrite

With this option checked, only the most recent tabs are saved, which is to say, only the tabs open at the time of the snapshot.

   Save pinned tabs

With this option checked, pinned tabs will be saved as well.
